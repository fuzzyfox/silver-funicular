# Assets

This directory contains the icons and logos for the Agent Usage Tracker extension.

## Required Files

### Extension Icon
- **icon.png** - Main extension icon (512x512px recommended)

### Agent Logos
- **claude-logo.png** - Claude/Anthropic logo (64x64px recommended)
- **junie-logo.png** - JetBrains Junie logo (64x64px recommended)
- **copilot-logo.png** - GitHub Copilot logo (64x64px recommended)

## Where to Get Logos

### Claude Logo
Download from Anthropic's brand assets or use their official logo from:
- https://www.anthropic.com/brand

### JetBrains Junie Logo
Download from JetBrains' brand assets or use their official logo from:
- https://www.jetbrains.com/company/brand/

### GitHub Copilot Logo
Download from GitHub's official logos or use their official logo from:
- https://github.com/logos

### Creating Icons
You can create simple placeholder icons using:
1. Online icon generators
2. Design tools like Figma, Sketch, or Canva
3. SVG to PNG converters

## Recommended Specifications
- Format: PNG with transparent background
- Extension icon: 512x512px
- Agent logos: 64x64px or higher
- Color mode: RGB
